# 第02章 if 与 switch 学习笔记

## 一、if语句

if用于条件判断，条件成立执行代码块。

### 关键点
- if
- else if
- else
- 条件表达式必须是逻辑表达式

---

## 二、逻辑判断

常用：
- >  <  >=  <=  ==
- &&（并且）
- ||（或者）

---

## 三、奇偶判断

num % 2 == 1 → 奇数  
num % 2 == 0 → 偶数

---

## 四、for循环（基础）

用于重复执行代码，例如输出1~100奇数。

---

## 五、switch语句

用于多分支固定值判断。

### 注意：
- case后必须加 break
- default是兜底逻辑

---

## 六、今天的问题

- 写代码时多个main函数导致结构混乱
- 大段代码没有拆分结构

---

## 七、总结

if负责范围判断  
switch负责固定值选择  
for负责循环结构
